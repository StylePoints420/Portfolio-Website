# Portfolio Website

**Welcome to my portfolio!** :wave:

## How to use this Project

1. Clone the project

```bash
git clone https://github.com/StylePoints420/Portfolio-Website.git
```

2. Install dependencies

```bash
npm i
```

3. Use in development

```bash
npm run dev
```

4. Use in production

```bash
npm run build
npm run serve
```

## Tech Stack

- **React**
- **Typescript**
- **Tanstack Router**
- **Tailwind CSS**
